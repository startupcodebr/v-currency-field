Vuetify Currency Field

Currency component for vuetify v-text-field

Get Started →

Works with vuetify 1.x and vuetify 2.x

Compatible with newer and older vuetify version.

No need update component with vuetify new props or listeners

The component dynamic binds the props and listeners to vuetify v-text-field

Many configuration options

Min, Max, Default Value, Allow Blank, Locale, Precision...
